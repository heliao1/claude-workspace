# claude-workspace

Espaço de trabalho pessoal para experimentos com [Claude Code](https://claude.ai/code).

## O que é isso?

Repositório usado como base para explorar funcionalidades do Claude Code, incluindo automações, scripts e integrações.

## Como usar

Clone o repositório e abra a pasta no Claude Code:

```bash
git clone https://github.com/heliao1/claude-workspace.git
cd claude-workspace
```
